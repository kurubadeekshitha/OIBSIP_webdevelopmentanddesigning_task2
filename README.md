# OIBSIP_WebDevelopment_Task3 – Personal Portfolio Website

## Objective
The objective of this task is to design and develop a personal portfolio website using HTML and CSS. The portfolio showcases personal details, skills, projects, and contact information in a visually appealing and structured layout. The goal is to create a professional-looking webpage that reflects creativity and technical skills.

## Steps Performed
1. Planned the layout and design for different sections of the portfolio, including the header, about, skills, projects, and contact sections.  
2. Created the structure of the webpage using HTML to organize content clearly.  
3. Styled the webpage using CSS to make it attractive, responsive, and easy to navigate.  
4. Focused on typography, color balance, and section spacing to maintain a professional design.  
5. Tested the website across different screen sizes to ensure responsiveness.  
6. Documented the project details in this README file.

## Tools & Technologies Used
- HTML5 – For webpage structure  
- CSS3 – For styling and responsive design  
- Visual Studio Code – Code editor used for development  

## Outcome
The final output is a fully responsive personal portfolio website that displays personal information, projects, and contact details in a clean and professional format. It effectively represents individual skills and creativity using front-end web development techniques.
